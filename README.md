# AureonVault

Administrador de archivos de escritorio hecho para JackStar.

## Enfoque

- App nativa para Windows con Python + Tkinter
- UI morado/dorado, resizable y pensada para escritorio
- Acceso a unidades, carpetas y archivos del PC
- Navegacion rapida, filtro por nombre y panel de detalle

## Ejecutar

```powershell
cd C:\Users\Jack\Documents\GitHub\Experimentos\AureonVault
python app.py
```

O con doble clic en:

`run_aureonvault.bat`

## Estructura

- `app.py`: aplicacion principal nativa
- `run_aureonvault.bat`: lanzador de Windows
- `prototype_web/`: primer MVP basado en servidor local, guardado como referencia
